# 付録：桁数変更対象一覧

現行帳票から桁数の変更がある項目一覧。一覧に記載がある項目は全帳票で変更反映をすること。

各項目は「現行会計 → 次期会計」の桁数変更を示す。表記：テーブル名（論理名）／物理名／名称／カラム名（物理名）／現行 桁数[byte]／現行 桁数[全角]／次期 桁数[全角]。

## 相手先マスタ（ATMT）

- **相手先名称1**（NM1）：現行 120byte・20全角 → 次期 60全角
- **相手先名称2**（NM2）：現行 120byte・20全角 → 次期 60全角
- **相手先略名**（SNM）：現行 48byte・8全角 → 次期 10全角

## 科目マスタ全般（YMSMT など）

- **科目名称**（NM）：現行 120byte・20全角 → 次期 21全角
- **科目略名**（SNM）：現行 48byte・8全角 → 次期 12全角

## 単価名称マスタ（TKNMMT）

- **単価略名**（SNM）：現行 48byte・8全角 → 次期 10全角

## 単価属性マスタ（TKZSMT）

- **単価規格略名**（TKKKSNM）：現行 48byte・8全角 → 次期 10全角

## 貯蔵品単価名称マスタ（CZTKNMMT）

- **単価略名**（SNM）：現行 48byte・8全角 → 次期 10全角

## 貯蔵品単価属性マスタ（CZTKZSMT）

- **単価規格略名**（TKKKSNM）：現行 48byte・8全角 → 次期 10全角

## 固定資産マスタ（KTSMT）

- **固定資産名称**（NM）：現行 120byte・20全角 → 次期 30全角

## 予算要求積算基礎（YYKSSKS）

- **積算基礎の名称と式**（SSKSNM）：現行 120byte・20全角 → 次期 40全角
  - 備考：文章も計算式も自由入力。

## 全般

- **金額**：現行 12byte・12全角 → 次期 14全角
  - 備考：カンマなし半角数値の桁数。
